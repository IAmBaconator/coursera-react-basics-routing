# coursera-react-basics-routing
An app to demonstrate the navigation bar in a single page load application.
## Goal
The goal is to show how SPA navigation loads a template rather the traditional page load.
# Lab Notes
Command to create a React app, `npx create-react-app counter_app` and start the browser preview of the app, `npm start`.
**Note:** The create-react-app command is deprecated.
Command to create _(this)_ React Router app, `npm install react-router-dom@6` and start the browser preview of the app, `npm start`.
